# Save all heatmap plots

Save all heatmap plots

## Usage

``` r
save_heatmaps(plot_list, output_dir = "heatmaps", width = 8, height = 6)
```

## Arguments

- plot_list:

  List of ggplot objects from plot_2d_heatmaps()

- output_dir:

  Directory to save plots

- width:

  Plot width in inches

- height:

  Plot height in inches
