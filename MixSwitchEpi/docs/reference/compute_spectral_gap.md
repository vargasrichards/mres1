# spectral gap calculation

compute the difference in absolute value between the dominant and
second-most dominant eigenvals. Should provide some measure of the time
of duration transient behaviour.

## Usage

``` r
compute_spectral_gap(pars)
```

## Arguments

- pars:

  parameters for model run
