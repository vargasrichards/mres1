# Run system with specified level of assortativity

This function is helpful in conducting parallelised parameter scans

## Usage

``` r
run_epsilon(epsilon, parameters)
```

## Arguments

- epsilon:

  assortativity parameter. has to be between 0 and 1. See (Garnett et
  al., 1999) for details on how this is interpreted .

- parameters:

## References

Garnett, Geoff P., et al. "The transmission dynamics of gonorrhoea:
modelling the reported behaviour of infected patients from Newark, New
Jersey." Philosophical Transactions of the Royal Society of London.
Series B: Biological Sciences 354.1384 (1999): 787-797.
