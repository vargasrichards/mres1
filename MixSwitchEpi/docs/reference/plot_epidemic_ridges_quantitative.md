# True ridgeline plot with quantitative y-axis and adjustable spacing

True ridgeline plot with quantitative y-axis and adjustable spacing

## Usage

``` r
plot_epidemic_ridges_quantitative(model_output, compartment = "I", spacing = 1)
```

## Arguments

- spacing:

  Vertical spacing between classes. \< 1 = more overlap, \> 1 = more
  separation
