# Forces proportionate (random) mixing in Haslemere activity classes Hence epsilon = 0

Forces proportionate (random) mixing in Haslemere activity classes Hence
epsilon = 0

## Usage

``` r
haslemere_mix_parms(N, n_activity, target_r0)
```

## Arguments

- N:

  the total population size

- n_activity:

  the number of activity classes

- target_r0:

  the target R0 value
