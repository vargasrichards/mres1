# Parameterically generate exponential mixing model

Automatically computes the normalisation constants needed

## Usage

``` r
exponential_mixing(n_activity, epsilon_assort, act_vector, class_size_vector)
```

## Arguments

- n_activity:

  the number of activity classes

- epsilon_assort:

  the assortativity of the contact pattern

- act_vector:

  the vector of activity class scores

- class_size_vector:

  the vector of activity class sizes
