# function which divides by the occupancy of each activity class

function which divides by the occupancy of each activity class

## Usage

``` r
rescale_vars(tidy_epi, num_classes, times)
```

## Arguments

- tidy_epi:

  the epidemic in tidy form

- num_classes:

  the number of activity classes being modelled

- times:

  the timesteps simulated. will be joined to the epi
