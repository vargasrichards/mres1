# Construct the contact matrix

Construct the contact matrix

## Usage

``` r
make_contact_matrix(
  n_activity,
  epsilon,
  activity_vector,
  class_size_vector,
  power_param = NULL,
  mixing_model
)
```

## Arguments

- n_activity:

  the number of activity groups in the model

- epsilon:

  the assortativity for the contact/mixing model

- activity_vector:

  the vector of activity scores

- class_size_vector:

  the vector of class sizes in the model

- power_param:

  the power parameter for

- mixing_model:

  the model of contact/mixing
