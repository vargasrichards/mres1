# Plot homogeneous mass action model

Plot homogeneous mass action model

## Usage

``` r
homogeneous_curve(r0)
```

## Arguments

- r0:

  the target r0 for the homogeneous model. Note that this uniquely
  defines the epidemic given we have fixed params

- n_activity:

  the number of activity classes involved.

## Value

a ggplot2 object and saves output to pdf file
